sshj by shikhar

SSH, SCP and SFTP for Java

SSHv2 library for Java

Dependencies

slf4j is required. bouncycastle is highly recommended and required for using some of the crypto algorithms. jzlib is required for using zlib compression.

Install

$ buildr install
